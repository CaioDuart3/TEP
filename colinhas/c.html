Manipulação de Strings

strlen

Obtém o comprimento de uma string.

int len = strlen(str);

strcpy e strncpy

strcpy copia uma string para outra; strncpy copia até n caracteres.

strcpy(dest, src);
strncpy(dest, src, n); // até n caracteres

strcmp

Compara duas strings e retorna 0 se forem iguais.

int cmp = strcmp(str1, str2);

strcat e strncat

strcat concatena uma string ao final de outra; strncat concatena até n caracteres.

strcat(dest, src);
strncat(dest, src, n);

Linked List

Estrutura de nó para uma lista encadeada simples e função para inserir um novo nó no início.

struct Node {
    int data;
    struct Node* next;
};

struct Node* head = NULL;
void insert(int new_data) {
    struct Node* new_node = (struct Node*) malloc(sizeof(struct Node));
    new_node->data = new_data;
    new_node->next = head;
    head = new_node;
}

Algoritmos de Ordenação

Bubble Sort

Algoritmo de ordenação que compara e troca elementos adjacentes.

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
        }
    }
    }
}

Quick Sort

Algoritmo de ordenação baseado em divisão e conquista, com uma função de particionamento.

int partition(int arr[], int low, int high) {
    int pivot = arr[high];
    int i = (low - 1);
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }
    }
    int temp = arr[i + 1];
    arr[i + 1] = arr[high];
    arr[high] = temp;
    return (i + 1);
}

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

Busca

Busca Linear

Busca um elemento em um array, verificando cada posição até encontrar.

int linearSearch(int arr[], int n, int x) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == x)
            return i;
    }
    return -1;
}

Busca Binária

Procura um elemento em um array ordenado dividindo-o em duas partes.

int binarySearch(int arr[], int l, int r, int x) {
    while (l <= r) {
        int m = l + (r - l) / 2;
        if (arr[m] == x) return m;
        if (arr[m] < x) l = m + 1;
        else r = m - 1;
    }
    return -1;
}
