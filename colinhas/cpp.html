Manipulação de Strings

Comprimento da String

Obtém o comprimento de uma string usando o método .length().

std::string str = "Exemplo";
int len = str.length();

Concatenar Strings

Concatena duas strings usando o operador + ou o método .append().

std::string str1 = "Olá, ";
std::string str2 = "mundo!";
std::string result = str1 + str2;
// ou
result.append(str2);

Comparar Strings

Compara duas strings com o operador == ou o método .compare().

if (str1 == str2) {
    // são iguais
}
int result = str1.compare(str2); // retorna 0 se forem iguais

Encontrar Substring

Encontra a posição da primeira ocorrência de uma substring.

std::string str = "Olá, mundo!";
size_t pos = str.find("mundo"); // pos = 5

Encontrar Substring (Reversa)

Encontra a posição da última ocorrência de uma substring.

size_t pos = str.rfind("o"); // pos = 7

Contar quantidade de vezes que um elemento aparece em um array

using namespace std int main() { vector vec = {1, 2, 2, 3, 2, 4, 5}; int count = count(vec.begin(), vec.end(), 2); cout << "O número 2 aparece " << count << " vezes no vetor." << std::endl; return 0; }

Funções Matemáticas

pow()

Calcula a potência de um número.

double result = pow(2, 3); // result = 8

sqrt()

Calcula a raiz quadrada de um número.

double result = sqrt(16); // result = 4

min()

Retorna o menor valor entre dois números.

int menor = min(10, 5); // menor = 5

max()

Retorna o maior valor entre dois números.

int maior = max(10, 5); // maior = 10

swap()

Troca os valores de duas variáveis.

int a = 10, b = 20;
std::swap(a, b); // a = 20, b = 10

gcd()

Calcula o máximo divisor comum entre dois números.

int result = std::__gcd(8, 12); // result = 4

toupper()

Converte um caractere para maiúscula.

char upper = toupper('a'); // upper = 'A'

tolower()

Converte um caractere para minúscula.

char lower = tolower('A'); // lower = 'a'

floor()

Arredonda um número para baixo.

double result = floor(3.7); // result = 3

ceil()

Arredonda um número para cima.

double result = ceil(3.2); // result = 4

Estruturas de Dados Comuns

Vector

Estrutura de dados dinâmica que permite adicionar e acessar elementos facilmente.

std::vector vec;
vec.push_back(10); // adiciona 10 ao final
int x = vec[0]; // acessa o primeiro elemento

Map

Estrutura de dados chave-valor, onde cada chave é única.

std::map idade;
idade["Alice"] = 25;
int idadeAlice = idade["Alice"];

Algoritmos de Ordenação

Sort (STL)

Usa a função std::sort para ordenar elementos em um container.

std::vector vec = {3, 1, 4, 1, 5};
std::sort(vec.begin(), vec.end());
// vec agora está ordenado

Reverse (STL)

Inverte a ordem dos elementos em um container.

std::reverse(vec.begin(), vec.end());

Busca

Busca Linear

Busca um elemento em um vetor verificando cada posição até encontrá-lo.

int linearSearch(const std::vector& vec, int x) {
    for (int i = 0; i < vec.size(); i++) {
        if (vec[i] == x)
            return i;
    }
    return -1;
}

Busca Binária

Procura um elemento em um vetor ordenado usando a função std::binary_search.

bool found = std::binary_search(vec.begin(), vec.end(), x);
