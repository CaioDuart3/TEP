Linguagem Python

Funções Úteis em Programação Competitiva

Zip (Método zip)

Usa a função zip para combinar dois ou mais iteráveis em uma lista de tuplas.


                    a = [1, 2, 3]
                    b = ['a', 'b', 'c']
                    zipped = list(zip(a, b))  # [(1, 'a'), (2, 'b'), (3, 'c')]

Counter (Classe collections.Counter)

A classe Counter conta a frequência de elementos em um iterável.

from collections import Counter
        
        data = ['a', 'b', 'c', 'a', 'b', 'a']
        counter = Counter(data)  # Counter({'a': 3, 'b': 2, 'c': 1})

Most Common (Método most_common)

Retorna as n entradas mais comuns de um Counter.

most_common_items = counter.most_common(2)  # [('a', 3), ('b', 2)]

Defaultdict (Classe collections.defaultdict)

Uma subclasse de dict que fornece um valor padrão para chaves inexistentes.

from collections import defaultdict
        
        d = defaultdict(int)  # valor padrão é 0
        d['a'] += 1  # d['a'] agora é 1

Set (Estrutura set)

Conjuntos permitem operações de união, interseção e diferença, além de remover duplicatas automaticamente.

a = {1, 2, 3}
        b = {2, 3, 4}
        union = a | b  # {1, 2, 3, 4}
        intersection = a & b  # {2, 3}
        difference = a - b # {1}

Itertools (Módulo itertools)

Fornece funções para criar iteradores eficientes, como permutations e combinations.

import itertools
        
        items = [1, 2, 3]
        perms = list(itertools.permutations(items))  # [(1, 2, 3), (1, 3, 2), ...]

Bisect (Módulo bisect)

Auxilia na busca e inserção em listas ordenadas.

import bisect
        
        sorted_list = [1, 3, 4, 6]
        bisect.insort(sorted_list, 5)  # A lista agora será [1, 3, 4, 5, 6]

Heapq (Módulo heapq)

Implementa uma fila de prioridade, permitindo obter os maiores ou menores elementos de forma eficiente.

import heapq
        
        numbers = [1, 3, 5, 2, 4]
        smallest = heapq.nsmallest(3, numbers)  # [1, 2, 3]

Math (Módulo math)

Fornece várias funções matemáticas úteis, como gcd e factorial.

import math
        
        gcd_value = math.gcd(18, 24)  # 6

Estruturas de Dados Comuns

Listas

Listas são estruturas de dados que permitem armazenar múltiplos itens em uma única variável. Elas são mutáveis, o que significa que os elementos podem ser alterados após a criação.

Criação de uma Lista

Você pode criar uma lista usando colchetes `[]` ou a função list().

lista1 = [1, 2, 3, 4]
            lista2 = list([5, 6, 7, 8])

Adicionar Elementos

Para adicionar um elemento ao final de uma lista, use o método .append(). Para adicionar múltiplos elementos, use .extend().

lista1.append(5)  # Agora lista1 é [1, 2, 3, 4, 5]
            lista1.extend([6, 7])  # Agora lista1 é [1, 2, 3, 4, 5, 6, 7]

Inserir Elemento em uma Posição Específica

Você pode inserir um elemento em uma posição específica usando .insert(index, elemento).

lista1.insert(2, 'novo')  # Insere 'novo' na posição 2

Remover Elementos

Para remover um elemento, você pode usar .remove() para remover pela primeira ocorrência do valor ou .pop() para remover um elemento pelo índice.

lista1.remove(3)  # Remove o primeiro 3
            elemento = lista1.pop(0)  # Remove e retorna o primeiro elemento

Acessar Elementos

Os elementos de uma lista podem ser acessados por meio de índices. O índice começa em 0.

primeiro = lista1[0]  # Acessa o primeiro elemento
            ultimo = lista1[-1]     # Acessa o último elemento

Iterar sobre uma Lista

Você pode usar um loop for para iterar sobre os elementos de uma lista.

for elemento in lista1:
                print(elemento)

Dicionários

Dicionários são estruturas de dados que armazenam pares chave-valor. Eles são mutáveis, o que significa que você pode alterar os valores associados a chaves existentes.

Criação de um Dicionário

Você pode criar um dicionário usando chaves `{}` ou a função dict().

dicionario = {"nome": "Alice", "idade": 30}
            dicionario2 = dict(endereco="Rua 123", cidade="São Paulo")

Acessar Valores

Os valores em um dicionário podem ser acessados usando suas chaves.

nome = dicionario["nome"]  # Acessa o valor associado à chave "nome"

Adicionar ou Alterar Elementos

Você pode adicionar um novo par chave-valor ou alterar o valor de uma chave existente simplesmente atribuindo um valor à chave.

dicionario["altura"] = 1.65  # Adiciona nova chave
            dicionario["idade"] = 31          # Altera o valor da chave "idade"

Remover Elementos

Para remover um par chave-valor, use del ou o método .pop().

del dicionario["idade"]  # Remove a chave "idade"
            idade = dicionario.pop("idade", "Chave não encontrada")  # Remove e retorna o valor da chave

Iterar sobre um Dicionário

Você pode iterar sobre as chaves, valores ou pares chave-valor de um dicionário usando um loop for.

for chave, valor in dicionario.items():
                print(chave, valor)

Verificar a Existência de uma Chave

Você pode verificar se uma chave está presente no dicionário usando a palavra-chave in.

if "nome" in dicionario:
                print("A chave 'nome' existe no dicionário")

Manipulação de Strings

Comprimento da String

Obtém o comprimento de uma string usando a função len().

str = "Exemplo"
len_str = len(str)

Concatenar Strings

Concatena duas strings usando o operador + ou .join().

str1 = "Olá, "
str2 = "mundo!"
result = str1 + str2
# ou
result = "".join([str1, str2])

Comparar Strings

Compara duas strings com o operador ==.

if str1 == str2:
    # são iguais

Dividir String

Divide uma string em uma lista de substrings usando o método .split().

frase = "Olá mundo em Python"
palavras = frase.split()  # ['Olá', 'mundo', 'em', 'Python']

Juntar Lista em String

Junta elementos de uma lista em uma única string usando o método .join().

palavras = ["Olá", "mundo", "em", "Python"]
frase = " ".join(palavras)  # "Olá mundo em Python"

Aplicar Função a Cada Elemento com map()

Aplica uma função a cada elemento de uma lista usando map().

numeros = [1, 2, 3, 4]
quadrados = list(map(lambda x: x**2, numeros))  # [1, 4, 9, 16]

Filtrar Elementos com filter()

Filtra elementos de uma lista que atendem a uma condição usando filter().

numeros = [1, 2, 3, 4, 5]
pares = list(filter(lambda x: x % 2 == 0, numeros))  # [2, 4]

Reduzir uma Lista com reduce()

Aplica uma função de acumulação a todos os elementos de uma lista usando reduce().

from functools import reduce

numeros = [1, 2, 3, 4]
produto = reduce(lambda x, y: x * y, numeros)  # 24

Conjuntos (sets)

Conjuntos são tipos de dados não ordenados em Python, representados por chaves `{}` e que não permitem elementos duplicados. Eles são úteis para operações de união, interseção e diferença.

Criação de um Conjunto

Você pode criar um conjunto usando chaves ou a função set().

conjunto1 = {1, 2, 3, 4}
        conjunto2 = set([3, 4, 5, 6])

Adicionar Elementos

Para adicionar um elemento a um conjunto, use o método .add().

conjunto1.add(5)  # Agora conjunto1 é {1, 2, 3, 4, 5}

Remover Elementos

Você pode remover um elemento com .remove() ou .discard(). A diferença é que .remove() gera um erro se o elemento não existir.

conjunto1.remove(5)  # Remove 5
        conjunto1.discard(10)  # Não gera erro, mesmo que 10 não esteja presente

Operações com Conjuntos

Os conjuntos suportam operações matemáticas como união, interseção e diferença.

uniao = conjunto1 | conjunto2  # União
        interseccao = conjunto1 & conjunto2  # Interseção
        diferenca = conjunto1 - conjunto2    # Diferença

Verificar a Existência de um Elemento

Você pode verificar se um elemento está em um conjunto usando a palavra-chave in.

if 3 in conjunto1:
            print("3 está no conjunto")

Busca

Busca Linear

Busca um elemento em uma lista verificando cada posição até encontrá-lo.

def busca_linear(lista, x):
    for i in range(len(lista)):
        if lista[i] == x:
            return i
    return -1

Busca Binária

Procura um elemento em uma lista ordenada usando a biblioteca bisect.

import bisect
pos = bisect.bisect_left(lista_ordenada, x)
if pos != len(lista_ordenada) and lista_ordenada[pos] == x:
    print("Elemento encontrado")
else:
    print("Elemento não encontrado")

Algoritmos de Ordenação

Sort (Método sorted)

Usa a função sorted para ordenar elementos em uma lista.

lista = [3, 1, 4, 1, 5]
lista_ordenada = sorted(lista)  # retorna uma nova lista ordenada

Reverse (Método reverse)

Inverte a ordem dos elementos de uma lista.

lista.reverse()
